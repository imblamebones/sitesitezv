# TechFix — Ремонт техники (Учебный проект)

Статичная HTML-страница, стилизованная с помощью **SCSS** по модульной архитектуре, демонстрирующая навыки верстки и использования препроекторов. 

**Тема проекта:** Сайт ремонта техники (услуги, прайс, заявка).

---

##  Как запустить компиляцию SCSS

В проекте используется `sass` через npm-скрипты.

1. Убедитесь, что у вас установлен [Node.js](https://nodejs.org/).
2. Скачайте репозиторий и откройте его в терминале.
3. **Установите зависимости (Sass):**
   ```bash
   npm install
Для разовой сборки CSS выполните:

Bash
npm run build
Эта команда выполнит: sass scss/styles.scss css/styles.css

Для запуска в режиме отслеживания изменений (Watch):

Bash
npm run watch
Эта команда выполнит: sass --watch scss/styles.scss:css/styles.css

Альтернативно: Вы можете использовать расширение Live Sass Compiler в VS Code, нажав кнопку "Watch Sass" в нижней панели редактора.


В проекте реализованы требования на оценки 3, 4 и дополнительные критерии на 5:


Проект успешно компилируется через SCSS → CSS.

Итоговый файл css/styles.css подключен в index.html.

Создана папка sections/ с разбиением стилей.

Реализовано более 4 секций (header, mainscreen, services, advantages, pricing, contact, footer).

_variables.scss содержит все нужные переменные (шрифты, цвета, отступы).

_mixins.scss содержит переиспользуемые миксины для медиазапросов и кнопок.

Используется активная вложенность SCSS.


В проекте 7 осмысленных секций.

Использовано 3 брейкпоинта ($bp-md, $bp-lg, $bp-xl), активно применяемых в медиазапросах.

Реализованы состояния: :hover, :focus, :disabled, .is-active, .is-error.

Применяются сетки Flexbox (шапка, подвал) и CSS Grid (преимущества, услуги, форма).


Генерация классов через @for: В _base.scss циклом сгенерированы утилитарные классы отступов (.gap-1 до .gap-5).

Генерация классов через @each: Сгенерированы цветовые модификаторы кнопок (.btn--primary, .btn--success) и бейджей (.badge--primary) из карты $theme-colors.

Набор кнопок через Mixin: В _mixins.scss написаны миксины @mixin button-base, @mixin button-variant и @mixin button-outline для масштабируемой системы кнопок.

Методология: Во всем проекте строго соблюдается БЭМ (__ и --).
